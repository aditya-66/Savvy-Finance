# SAVVY - Finance Dashboard

A stunning, real-time personal finance dashboard and Telegram Bot integration.

## Features
- **Dynamic Dashboard**: Track expenses and savings with modern, responsive UI.
- **Budget Health**: Get an instant grade on your monthly financial health.
- **Telegram Bot**: Add expenses and income on-the-go directly from Telegram.

## Setup Instructions

### 1. Prerequisites
- Node.js installed
- MySQL Server installed and running

### 2. Installation
1. Clone this repository.
2. Run 
pm install to install all dependencies.
3. Create a .env file in the root directory and copy the contents of .env.example into it.
4. Fill in your actual MySQL database credentials and Telegram Bot Token in the .env file.

### 3. Database Setup
Run the SQL queries found in init.sql in your MySQL database to create the required tables.

### 4. Running the App
1. Start the server by running 
ode server.js
2. Open your browser and go to http://localhost:3000

Enjoy tracking your finances with SAVVY!
